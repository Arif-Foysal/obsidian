
#### Make a directory-
##### Windows
`mkdir directory_name` 
Make a directory that has spaces in between-
`mkdir 'directory name'`
or, seperating with a backtick
```
mkdir directory` name
```
##### Linux
Making directory-- same as powershell
Making directory with a space-
`mkdir 'directory name'`
or
`mkdir directory\ name`
