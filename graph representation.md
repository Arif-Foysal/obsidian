1. [[adjacency matrix]]
2. [[adjacency list]]

![[adjacency list]]

### Adjacency Matrix vs Adjacency List

| Action           | Adjacency Matrix | Adjacency List |
| ---------------- | ---------------- | -------------- |
| Adding Edge      | O(1)             | O(1)           |
| Removing an edge | O(1)             | O(N)           |
| Initializing     | O(N*N)           | O(N)           |
