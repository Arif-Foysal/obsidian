## what is standard template library?
standard template library is a set of c++ template classes to provide common programming data structures and functions such as-

- [[stack in c++]]
- [[queue in c++]]
- [[array in c++]]
- [[vector in c++]]
- [[lists in c++]]
- [[map and set in c++]]
- algorithms (sort, find, count if)
