#networking #computernetworks 

## Transmission mode

- Simplex
- Half Duplex
- Full Duplex

### Simplex
Unidirectional communication / one way communication.
### Half-Duplex
Each station can transmit and receive. But not at the same time.
e.g. walkie talkie
### Full-Duplex
Both stations can transmit and receive simultaneously.
e.g. telephone
