To delete a database, use `DROP DATABASE`
```sql
DROP DATABASE `classicmodels`;
```
The database will no longer show up in the list of databases.
```sql
SHOW DATABASES;
```

Output:
```sql
+--------------------+
| Database           |
+--------------------+
| information_schema |
| mysql              |
| performance_schema |
| sys                |
+--------------------+
4 rows in set (0.01 sec)
```
